Welcome to VOXCE.

VOXCE™ is the first open source clinical trial management framework and CTMS to offer truly complete control of data collection, operations and analysis, providing the ability to:

  • Manage patients from enrollment to withdraw
  • Conduct multicenter clinical trials and research projects from inception to market or bedside on a global scale
  • Manage data operations
  • Capture submission process to different submission organizations
  • Track multiple studies across sponsors and sites with diverse colleagues
  • Manage payments throughout a study
  • Track protocol changes
  • Manage different user groups
  • Create monitoring reports
  • Capture Adverse Events
  • Capture laboratories, laboratory data, drugs and which laboratories associated with sites and studies

VOXCE is built using a subscription model allowing multiple users to be part of that subscription, which allows users to utilize the same libraries and templates, which utilizes questions, sections and forms. An organization can have multiple subscriptions, where each subscription group will have their own libraries, questions and more.

Rather Drive Stick?

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
